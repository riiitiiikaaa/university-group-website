4. Reflection on Validation

Some images were missing alt attributes, and a few file names had spaces, causing src errors. I also found a few unclosed tags and incorrect nesting.

I added descriptive alt text for all images, renamed files to remove spaces, and corrected any unclosed or misnested tags. Then I re-validated each page with the W3C tool.

Validation taught me the importance of semantic HTML, proper file naming, and careful coding. It helps make pages accessible, error-free, and easier to maintain.
